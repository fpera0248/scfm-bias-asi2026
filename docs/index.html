Demographic Bias in Single-Cell Foundation Models

Evaluation and mitigation across models and cohorts — reproduce every result on your own GPU with prebuilt containers.

Code on GitHub Container guide Full README

What this is

We measure how well scFoundation, Geneformer, and scGPT recover cell types for underrepresented demographic groups, and test scDesign3 synthetic augmentation as a mitigation. Three cohorts (interstitial lung disease, colorectal cancer, and the Asian Immune Diversity Atlas), three demographic axes (ethnicity, sex, age), measured with iLISI and worst-group cell-type macro-F1.

This page points you at the code and the ready-to-run containers. The datasets and model weights keep their own licenses and are downloaded separately.

Reproduce on your own GPU

Each model ships as a public Docker image. Pull the one you want, mount your own .h5ad data, and run — nothing to build.

scFoundation GPU

Embedding + evaluation.

docker pull ghcr.io/fpera0248/scfm-scfoundation:latest

Geneformer GPU

Embedding + evaluation.

docker pull ghcr.io/fpera0248/scfm-geneformer:latest

scGPT GPU

Embedding + evaluation.

docker pull ghcr.io/fpera0248/scfm-scgpt:latest

scDesign3 CPU

Synthetic augmentation (R).

docker pull ghcr.io/fpera0248/scfm-scdesign3:latest

Run one

docker run --gpus all -it -v /path/to/your/data:/data \
    ghcr.io/fpera0248/scfm-scgpt:latest

# inside the container:
conda run -n scgpt310 python step2a_embed_scgpt_ethnicity.py
On HPC without Docker, the same images run under Apptainer: apptainer pull scfm-scgpt.sif docker://ghcr.io/fpera0248/scfm-scgpt:latest then apptainer run --nv -B /path/to/data:/data scfm-scgpt.sif.

What you need

  • Docker and the NVIDIA Container Toolkit on your machine.
  • An NVIDIA GPU + driver ≥ 520 — the images are CUDA 11.8 and carry their own CUDA libraries; the driver is the only host-side piece. The scDesign3 image is CPU-only.
  • Your own .h5ad data with an expression matrix, a cell_type column, and a demographic column (self_reported_ethnicity, sex, or age).

Mount your data at /data and point each step script's BASE variable at /data.
